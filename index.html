🛡️ Twitch ModPanel

Панель модератора для Twitch — быстрый доступ к инструментам управления стримом без переключения на Mod View.

📦 GitHub 📥 Установить Tampermonkey

Установка

Способ 1: Tampermonkey

  1. Установи Tampermonkey
  2. Нажми на кнопку "Установить Tampermonkey" выше
  3. Подтверди установку в окне Tampermonkey
  4. Зайди на любой стрим где ты модератор
  5. Нажми на кнопку "Панель модератора" над чатом

Способ 2: Chrome Extension

  1. Открой chrome://extensions/
  2. Включи Developer mode
  3. Нажми Load unpacked
  4. Выбери папку twitch-mod-panel
