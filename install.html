🛡️ Twitch ModPanel

Установка через Tampermonkey

  1. Установи Tampermonkey
  2. Нажми кнопку ниже
  3. Нажми Install
📦 Установить скрипт

Ручная установка:

  1. Скачай twitch-modpanel.user.js
  2. Открой Tampermonkey → Create new script
  3. Вставь содержимое файла
  4. File → Save

Использование

  1. Открой любой стрим на Twitch
  2. Над чатом появится кнопка "Панель модератора"
  3. Нажми для открытия панели
  4. Первый вход через меню Tampermonkey → "🔐 Войти"
